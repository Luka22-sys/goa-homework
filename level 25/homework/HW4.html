Inline

Inline გულისხმობს, რომ კოდი (JavaScript ან CSS) იწერება პირდაპირ HTML ელემენტში, ან HTML-ის შიგნით, ელემენტებთან ერთად. Inline JavaScript: კოდი ჩნდება პირდაპირ HTML ატრიბუტებში, მაგალითად onclick, onmouseover და სხვა. Inline CSS: სტილები ჩნდება პირდაპირ HTML ელემენტებში style ატრიბუტში

Internal

სწრაფად გამოიყენება და გამოსაყენებლად ძალიან მარტივია. ეს მეთოდი კარგია, როდესაც კოდში მხოლოდ მცირე ცვლილებებია საჭირო. ნაკლული: არ არის სასურველი დიდი პროექტებისთვის, რადგან ეს ართულებს კოდის მართვას და გაზრდის კოდს. ძალიან რთული ხდება სკალირების და დუბლირების პრობლემების მართვა. 2. Internal (შიდა) Internal ნიშნავს, რომ კოდი (JavaScript ან CSS) დაწერილია იმავე HTML დოკუმენტის შიგნით, მაგრამ არა ელემენტებში, არამედ ტეგებში. Internal JavaScript: JavaScript კოდი მდებარეობს ტეგში HTML დოკუმენტის შიგნით Internal კოდი მდებარეობს ტეგში HTML დოკუმენტის შიგნით.

external

3. External (გარე) External ნიშნავს, რომ JavaScript და CSS კოდი განთავსებულია ცალკე ფაილებში, რომლებიც უკავშირდება HTML დოკუმენტს. External JavaScript: JavaScript კოდი ცალკე .js ფაილშია, რომელიც ჩაეშვება HTML დოკუმენტში ტეგის მეშვეობით. External CSS: CSS კოდი ცალკე .css ფაილშია, რომელიც ჩაეშვება HTML დოკუმენტში ტეგის მეშვეობით.
